## Build a bitmap

### Modules
#### greyscale.js
##### Turn bitmap to greyscale

#### pixelate.js
##### Pixelates a bitmap

#### inversion.js
##### Inverts the colors of a bitmap

#### readfile.js
##### Reads the bitmap file

#### writefile.js
##### Writes buffer data to newly created file

#### validate-bmp.js
##### Validates the file types for each module

#### Running the app
* `npm start`
  
#### Tests
* NPM run tests
